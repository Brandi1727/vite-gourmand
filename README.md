# Vite & Gourmand

## Description

Projet réalisé dans le cadre de l'ECF Développeur Web et Web Mobile.

Application développée en PHP avec une base de données MySQL.

---

## Technologies utilisées

- PHP
- HTML
- CSS
- MySQL
- phpMyAdmin
- XAMPP

---

## Installation

1. Installer XAMPP.
2. Copier le dossier `vite-gourmand` dans :

C:\xampp\htdocs\

3. Démarrer Apache et MySQL depuis XAMPP.

4. Ouvrir phpMyAdmin.

5. Importer le fichier :

vite_gourmand.sql

6. Vérifier que les informations de connexion dans :

config/database.php

correspondent à votre configuration.

7. Ouvrir le navigateur :

http://localhost/vite-gourmand
